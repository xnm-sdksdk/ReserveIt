# ReserveIt

ReserveIt is a web application designed to facilitate the reservation of resources such as meeting rooms, equipment, or event spaces. It provides an intuitive interface for users to check availability, make reservations, and manage their bookings efficiently.

# Build Java Package with Maven

`./mvnw clean package -Dquarkus.package.type=uber-jar`

# Using docker compose

- To Build the compose file
  `docker compose build`

- To Run the compose file
  `docker compose up`

- To Stop the compose file
  `docker compose down`

# Booking Service GraphQL UI

- The GraphQL UI can be accessed from http://localhost:8080/q/graphql-ui/ .
